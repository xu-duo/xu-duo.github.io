用Hexo搭建自己的博客

2017-02-08

写下这篇文章记录一下我的博客搭建过程及遇到的问题，开始吧。

Github

首先要有自己的 github 账号，并在 github 上创建一个仓库，名为 xxx.github.io。

Node.js

其次是在电脑上安装有 Node.js 和 git，我使用的电脑是 windows 系统，Node.js 的安装及使用教程参考官网 http://nodejs.cn。Git 的安装及使用教程参考廖雪峰的教程 Git教程链接。或者参考hexo的官方文档 https://hexo.io/zh-cn/docs。

Hexo

下载安装 hexo，官方文档链接同上。安装完成之后，会在本地生成一个 Hexo 文件夹。选择主题，去 hexo 的主题网站https://hexo.io/themes选择自己喜欢的主题。我选择的是 Hacker 主题，从 github 上下载到本地，放到 hexo 文件夹 themes 下，然后根据自己的需求，进行修改。

出现的问题

我是用的是 windows 系统，在搭建博客过程中出现了如下问题：

通过尝试，得出以下解决方案：设置 git 的 path
先在 git 的安装目录中找到 xxx/git/bin 及 xxx/git/libexec/git-core 文件。(xxx 为 git 安装盘目录)
再在 计算机右键-属性-高级系统设置-环境变量-系统变量-path-编辑-新建
将以上两个路径复制添加到变量中，点击确定；重启电脑，重新运行，后提交，成功！！！
